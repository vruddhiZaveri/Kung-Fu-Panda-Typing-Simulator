# Kung Fu Panda Typing Simulator 🐼⌨️

A fast-paced typing game inspired by *Kung Fu Panda*, built during a 2-hour SPIT game development hackathon using AI-assisted development with Gemini Pro.

## 🎮 About the Game

Master Shifu throws dumplings with letters on them, and Po has to catch them.  
The player must type the correct letters before they hit Po. If the player misses, Po gets hit!

As the game continues, the speed increases and the letter combinations become more difficult.

## 🚀 Features

- Fun Kung Fu Panda inspired theme
- Letter-based typing gameplay
- Increasing difficulty level
- Score tracking
- Fast reaction and accuracy challenge
- Simple and interactive UI

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

## 🎯 How to Play

1. Start the game.
2. Watch the letters on the falling dumplings.
3. Type the correct letter on your keyboard.
4. Score points by catching dumplings.
5. Avoid missing letters and try to beat your high score!

## 📌 Project Info

This game was created as part of a 2-hour game development hackathon at SPIT, where participants had to build a game based on an animated movie theme.
